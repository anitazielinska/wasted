# Wasted

Drinking simulator, written in modern OpenGL.

## Controls

- W/A/S/D - movement
- SPACE/LSHIFT - fly up/down
- M - toggle flying
- R - reload shaders
- escape - deactivate game window
- left click on window - activate game window
- mouse - look around

### Requirements

These can be installed from your distribution repository.

- GLFW
- GLEW
- GLM
- Assimp
- DevIL

## How to run

```
make run
```
